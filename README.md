# 電気＋集客ツール - ランディングページ

電力供給サービスと集客ツールを統合した、ビジネス向けソリューションのランディングページです。

## 🎯 プロジェクト概要

このWebサイトは、電力コスト削減とデジタルマーケティングツールを組み合わせた革新的なサービスを紹介するランディングページです。トリドリマーケティング（https://top-marketing.toridori.me/）のようなシンプルで洗練されたカードデザインを採用しています。

## ✨ 主な機能

### 実装済み機能

1. **レスポンシブデザイン**
   - モバイル、タブレット、デスクトップに完全対応
   - モバイルメニュー（ハンバーガーメニュー）実装

2. **セクション構成**
   - ヒーローセクション：グラデーション背景で視覚的にインパクトのある第一印象
   - サービス紹介：**画像付きカード**で3つの主要サービスを紹介
   - 特徴セクション：**横型画像カード**で4つの選ばれる理由を説明
   - 料金プラン：3段階のプランをカード形式で表示
   - CTAセクション：無料相談への誘導
   - フッター：企業情報とリンク

3. **インタラクティブ機能**
   - スムーススクロール（ページ内リンク）
   - モーダルウィンドウ（お問い合わせフォーム）
   - スクロールアニメーション（カードのフェードイン）
   - ヘッダーの影表示（スクロール時）
   - 通知システム（フォーム送信時）
   - **画像ホバーエフェクト**（拡大アニメーション）

4. **お問い合わせフォーム**
   - 会社名、氏名、メール、電話番号、プラン選択、メッセージ
   - Google Apps Script APIへのデータ送信機能
   - バリデーション機能
   - 送信完了通知

5. **デザイン要素**
   - **画像を使用したカードレイアウト**（トリドリマーケティング風）
   - 柔らかい影効果
   - ホバーアニメーション
   - グラデーション背景
   - Font Awesome アイコン
   - Noto Sans JP フォント
   - **Unsplash高品質画像**使用
